## Setting Up Your Development Environment

1. Install Python and pip.
2. Set up a virtual environment:

   ```bash
   python -m venv venv
   source venv/bin/activate
   ```

3. Install development dependencies:

   ```bash
   pip install -r requirements-dev.txt
   ```
